# devops-netology
Readme

Игнорируется все содержимое каталогов .terraform в любых каталогах
**/.terraform/*

Игнорируются файлы
*.tfstate
*.tfstate.*

Игнорируются файлы
crash.log
crash.*.log

Игнорируются файлы
*.tfvars
*.tfvars.json

Игнорируются файлы
override.tf
override.tf.json
*_override.tf
*_override.tf.json

Игнорируются файлы
.terraformrc
terraform.rc
